
Для чего и как:
Программа поможет систематизировать имена и года выпусков всех книг которые нужно прочитать.
Предусмотрены такие операции с книгами как: Добавить, Удалить, Поиск, Бомбёжка, Заменить.
Все книги хранятся в удобном к чтению даже человеком формате

Сохранение в файл:
В последних версиях добавлено сохранение списка книг в файл JSON,
благодаря которому список не потеряется даже после закрытия программы.
Файл можно сохранить и в дальнейшем передать кому-либо / опубликовать.

Кто может использовать:
Учителя, ученики, люди которые любят читать, библиотекари...
